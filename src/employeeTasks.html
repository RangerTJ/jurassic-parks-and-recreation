Employee Tasks

Add New Employee Task

If you would like to add a new Employee Task to the database. Enter values for its attributes below and click the "Add Task" button. Employee tasks are entities that handle the many-to-many relationship between a given Employee and an open Task Assignment. Each entity represents a chunk of work contributed by a specific employee towards the completion of the matching task. Many employees can contribute to the same task, and a single employee can contribute to the same task multiple times (for different hourly contributions at different time intervals).
